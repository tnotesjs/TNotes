## 2. 📒 了解什么是数据库

**数据库（Database）** 是存储和管理数据的仓库。它不仅提供对数据的高效存取功能，还支持多用户并发访问，并保证数据的安全性和一致性。

### 2.1. 数据库的特点：

- **结构化存储**：数据按照一定的格式进行组织。
- **高效查询**：通过 SQL 语言可以快速检索、更新、删除数据。
- **安全性高**：支持权限控制，保障数据安全。
- **支持事务**：确保多个操作要么全部成功，要么全部失败。

> 📌 示例：一个学校的学生成绩管理系统中，学生信息、课程信息、成绩等都可以存储在数据库中，便于统一管理。

## 3. 📒 掌握什么是表、数据类型和主键

### 3.1. 表（Table）

**表是数据库中存储数据的基本单位**，由若干行（记录）和列（字段）组成。

- 每一行代表一条记录；
- 每一列代表一个字段（属性）。

> 📌 示例：`students` 表可能包含 `id`、`name`、`age`、`gender` 等字段。

---

### 3.2. 数据类型（Data Type）

**数据类型决定了该字段能存储什么类型的数据**，例如整数、字符串、日期等。

常见的数据类型包括：

- 数值型：`INT`, `FLOAT`, `DECIMAL`
- 字符串型：`CHAR`, `VARCHAR`, `TEXT`
- 日期时间型：`DATE`, `DATETIME`, `TIMESTAMP`
- 布尔型：`BOOLEAN`

> 📌 示例：`age` 字段使用 `INT` 类型；`name` 使用 `VARCHAR(50)`。

---

### 3.3. 主键（Primary Key）

**主键是唯一标识表中每条记录的字段或字段组合**，具有以下特点：

- **唯一性**：不能有重复值
- **非空性**：不能为 NULL
- 通常用于建立表之间的关系

> 📌 示例：`id` 字段作为主键，每个学生的 `id` 都是唯一的。

---

## 4. ✅ 熟悉数据库的技术构成

数据库系统一般由以下几个部分组成：

| 组成部分 | 功能说明 |
| --- | --- |
| **数据库管理系统（DBMS）** | 负责数据库的创建、管理和维护 |
| **SQL 语言** | 结构化查询语言，用于操作数据库 |
| **数据库应用程序接口（如 JDBC、ODBC、MySQL Connector）** | 支持程序连接数据库并进行交互 |
| **数据库服务器** | 提供数据库服务，处理客户端请求 |
| **数据库客户端工具** | 如 MySQL Workbench、Navicat，用于图形化操作数据库 |

---

## 5. 🤔 什么是 MySQL？

**MySQL 是一个开源的关系型数据库管理系统（RDBMS）**，主要用于 Web 应用开发，广泛应用于中小型网站和企业级应用中。

### 5.1. MySQL 的主要特点：

- **开源免费**：社区版完全免费，适合学习和小型项目。
- **高性能**：支持高并发访问，响应速度快。
- **跨平台支持**：支持 Windows、Linux、macOS 等多种操作系统。
- **易用性强**：安装配置简单，文档丰富。
- **安全性高**：支持用户权限管理、SSL 加密等安全机制。

> 🧩 MySQL 被 Facebook、Twitter、YouTube、阿里巴巴等大型互联网公司广泛使用。

---

## 6. ✅ 掌握常见的 MySQL 工具

### 6.1. MySQL 自带工具

| 工具名称             | 功能说明                                     |
| -------------------- | -------------------------------------------- |
| `mysql` 命令行客户端 | 执行 SQL 语句，查看结果                      |
| `mysqldump`          | 备份数据库                                   |
| `mysqladmin`         | 管理 MySQL 服务器（如启动/停止、修改密码等） |
| `mysqlimport`        | 导入文本文件数据到数据库                     |

---

### 6.2. 图形化管理工具

| 工具名称              | 功能说明                                         |
| --------------------- | ------------------------------------------------ |
| **MySQL Workbench**   | 官方推荐工具，支持建模、SQL 开发、管理等功能     |
| **Navicat for MySQL** | 强大的图形界面工具，支持远程连接、数据同步       |
| **phpMyAdmin**        | 基于 PHP 的网页管理工具，常用于 Web 项目后台管理 |
| **DBeaver**           | 支持多种数据库的通用数据库管理工具               |
